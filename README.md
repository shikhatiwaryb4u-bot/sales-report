# Project Title: Sales Analysis of Laptops and Accessories Across India
This project focuses on analyzing sales data of laptops and their accessories across different regions of India. The primary objective was to extract meaningful insights from raw data to understand sales performance, customer trends, and product demand.

The project began with data cleaning and preprocessing in Microsoft Excel, where missing values were handled, inconsistent entries were corrected, and data was structured properly for analysis. Techniques such as removing duplicates, filling blanks, and standardizing formats were applied to ensure data accuracy and reliability.

After preparing the dataset, the cleaned data was imported into Power BI for visualization and analysis. Interactive dashboards were created to present key insights, including:

Total sales and revenue trends over time
Region-wise and state-wise sales performance
Best-selling laptop models and accessories
Category-wise contribution to overall sales
Monthly and yearly sales trends

Advanced Power BI features such as DAX (Data Analysis Expressions) were used to create calculated columns and measures for deeper analysis. Visual elements like bar charts, line graphs, pie charts, and slicers were used to make the dashboard dynamic and user-friendly.

This project demonstrates strong skills in data cleaning, data transformation, and data visualization, along with the ability to convert raw data into actionable business insights that can support decision-making.
